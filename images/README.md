# Images

This directory contains all visual assets used in the **Automatic Battery Monitoring System using CAN Bus and Node-RED** repository.

The images are organized into dedicated folders to separate architecture diagrams, flowcharts, hardware illustrations, dashboard screenshots, CAN protocol documentation, and concept results.

> **Note**
>
> Images representing dashboard output, UART output, CAN traffic, or monitoring screens are clearly identified as **Concept Illustrations** unless they are captured from a real implementation.

---

# Image Guidelines

All graphics in this repository follow these principles:

- Professional engineering style
- White background where applicable
- High-resolution PNG images
- GitHub-friendly formatting
- Consistent naming conventions
- Optimized for documentation

---

# Naming Convention

Use lowercase filenames with underscores.

Examples:

```
system_architecture.png
hardware_block_diagram.png
can_frame_layout.png
node_red_dashboard.png
concept_dashboard.png
```
