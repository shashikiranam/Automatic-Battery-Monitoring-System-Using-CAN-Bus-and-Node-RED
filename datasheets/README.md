# Datasheets

This folder is a placeholder for component datasheets referenced by the
hardware documentation. Datasheets are copyrighted by their respective
manufacturers and are not redistributed in this repository — download the
official versions from the links below and place them here for offline
reference.

| Component | Manufacturer     | Datasheet Source                                              |
|-----------|-------------------|-------------------------------------------------------------------|
| ESP32-WROOM-32 | Espressif     | https://www.espressif.com/en/products/socs/esp32               |
| MCP2515        | Microchip     | https://www.microchip.com/en-us/product/MCP2515                |
| TJA1050        | NXP           | https://www.nxp.com/products/TJA1050                            |
| INA219         | Texas Instruments | https://www.ti.com/product/INA219                          |
| DS18B20        | Analog Devices (Maxim) | https://www.analog.com/en/products/ds18b20.html        |
| HX711          | Avia Semiconductor | (search "HX711 datasheet")                                 |

## Suggested Naming Convention

When you add a datasheet PDF to this folder, name it:

```
<manufacturer>_<part-number>_datasheet.pdf
```

e.g. `microchip_mcp2515_datasheet.pdf`
